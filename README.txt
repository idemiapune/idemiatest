You have a sample REST API for a resource named 'Person'. 

Follow the 4 classes related to Person:
Person.java
PersonController.java
PersonSrvc.java
PersonRepository.java

These 4 classes are enough to create a RestFul API which can do CRUD 
on 'Person' resource.

How to run the project:

Right click on the project and Run As -> Spring Boot App

How to see the Database:

In the docking bar of the MacBook window there's a tool called Sequel Pro. 
That's the client for MySQL database.

How to test the RestFul API:

In the docking bar of the MacBook window there's a tool called Postman. 
It has got some samples for executing RestFul APIs related to Person.
